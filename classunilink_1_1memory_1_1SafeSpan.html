Detailed Description

template<typename T, std::size_t Extent = std::dynamic_extent>
class unilink::memory::SafeSpan< T, Extent >

C++20 std::span based memory view.

SafeSpan is now a specialized version of std::span that maintains backward compatibility for at() and ensures subspan returns SafeSpan.
