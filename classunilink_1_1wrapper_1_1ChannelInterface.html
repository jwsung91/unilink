virtual bool unilink::wrapper::ChannelInterface::send ( std::string_view  data)
pure virtual

Enqueue data for transmission.

Returns
true Data was accepted into the send queue.
false Data was dropped — channel not connected or backpressure limit reached.

The call is non-blocking. Delivery is not guaranteed even when true is returned; network errors are reported via on_error().

◆ stop()

virtual void unilink::wrapper::ChannelInterface::stop ( )
pure virtual

Stop the channel and block until all pending async operations are cancelled.

Safe to call from any thread. After stop() returns, no further callbacks will fire and it is safe to destroy the object. Calling stop() more than once is a no-op.

Implemented in unilink::wrapper::UdsClient, unilink::wrapper::Udp, unilink::wrapper::TcpClient, and unilink::wrapper::Serial.
