# Assets

This directory contains project identity assets used by the core repository README, GitHub social preview, and related project materials.

Documentation content images live in `unilink-docs`.
Benchmark result images live in `unilink-benchmarks`.
Container workflow images live in `unilink-containers`.

Core assets are limited to project identity images, such as README logos and the repository social preview image. Full documentation images, generated Doxygen assets, tutorial screenshots, benchmark result charts, and repo-specific workflow diagrams should remain in their owning repositories.
