bool unilink::wrapper::UdpClient::send ( std::string_view  data)
overridevirtual

Enqueue data for transmission, honouring the backpressure strategy.

BestEffort: non-blocking, drops if the send queue is full. Reliable: blocks until queue pressure is relieved, then enqueues.

Returns
true Data was accepted into the send queue.
false Data was dropped (not connected, or BestEffort queue full).

◆ stop()

void unilink::wrapper::UdpClient::stop ( )
overridevirtual

Stop the channel and block until all pending async operations are cancelled.

Safe to call from any thread. After stop() returns, no further callbacks will fire and it is safe to destroy the object. Calling stop() more than once is a no-op.
